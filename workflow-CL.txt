Workflow CI/CD Project Machine Learning

1. Data preprocessing dilakukan menggunakan script automate_Riyana.py yang mengubah dataset mentah menjadi dataset siap latih.
2. Dataset hasil preprocessing disimpan sebagai train_preprocessed.csv dan test_preprocessed.csv.
3. Model machine learning dilatih menggunakan script modelling.py dengan algoritma RandomForestClassifier.
4. Proses training model dicatat menggunakan MLflow Tracking untuk menyimpan parameter, metrics, dan artifacts model.
5. Model kemudian disajikan (serving) menggunakan FastAPI melalui script inference.py.
6. Monitoring sistem dilakukan menggunakan Prometheus untuk mengumpulkan metrik seperti jumlah request, latency prediksi, dan penggunaan CPU.
7. Visualisasi monitoring dilakukan menggunakan Grafana dashboard.
8. Sistem alerting dibuat menggunakan Grafana Alert Rules untuk mendeteksi kondisi tertentu pada metrik monitoring.
